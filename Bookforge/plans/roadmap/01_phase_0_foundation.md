# Phase 0 – Foundation

Goal:
Create the core Bookforge project skeleton and execution framework.

This phase establishes:

- project structure
- configuration system
- artifact storage
- schema definitions
- Ollama client
- CLI entrypoint
- validation gates
- testing harness

No real story generation occurs in this phase.

Acceptance criteria:

- Bookforge package installs
- CLI command executes
- config loads successfully
- schemas validate
- gate runner executes
- smoke tests pass

This phase ensures the system foundation is stable before creative generation begins.
