# Bookforge Overview

Bookforge generates fiction through staged decomposition.

The system transforms a structured brief into progressively detailed artifacts:

Project Brief
→ Story Skeleton
→ Beat Outline
→ Chapter Plan
→ Scene Packets
→ Draft Sections
→ Chapters
→ Final Manuscript

Each stage must pass validation gates before advancing.

Key design goals:

- Reproducible outputs
- Strong narrative structure
- Canon compliance
- Controlled creative expansion
- Artifact-based workflow
- Human review capability

Bookforge is local-first and initially targets Ollama models.
