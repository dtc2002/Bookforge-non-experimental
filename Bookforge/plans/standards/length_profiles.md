# Length Profiles

Bookforge uses length targets to constrain planning scale.

## Profiles

short_story
- target range: 3000 to 8000 words
- expected chapters: 1 to 5

novella
- target range: 20000 to 50000 words
- expected chapters: 8 to 20

novel
- target range: 70000 to 120000 words
- expected chapters: 20 to 60

## Rules

Planning should remain proportionate to selected length target.

The system should flag:
- excessive chapter count for target length
- too few chapters for declared scope
- beat density that does not match length target
