# Artifact Structure

All generated artifacts must be stored under:

Bookforge/data/projects/<project_id>/

Example layout:

Bookforge/data/projects/PROJECT123/

    project_manifest.json

    brief/
        brief_v1.json

    skeleton/
        story_skeleton_v1.json

    beats/
        beat_outline_v1.json

    chapters/
        chapter_plan_v1.json

    scenes/
        scene_packets_chapter_1_v1.json

    drafts/
        chapter_1_draft_v1.md

    validation/
        planning_review_v1.json
        pressure_audit_v1.json
        payoff_audit_v1.json
