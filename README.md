# Universe Bible Tool

A comprehensive tool for managing universe bible data with Ollama integration

## Features
- Dataclass-based project configuration
- Ollama HTTP API integration
- SQLite database for storing characters, plot threads, and canon facts
- Typer-based CLI for managing your universe bible
- Structured logging and data validation

## Getting Started

1. Clone the repository
2. Install dependencies: `poetry install`
3. Initialize the database: `poetry run init`
